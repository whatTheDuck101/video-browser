# video-browser

A video browsing app that uses vue files to create vue components.
Vue components
SearchBar
VideoList
VideoListItem
Video Detail
App

With a simple design style.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
